## Time Calculation System

The galactic time system uses real astronomical data where available:
- **Day Length**: Based on actual rotation periods in hours
- **Year Length**: Based on orbital periods around their star/planet
- **Time Display**: Shows local time calculated from Earth time using day length ratios
- **Calendar Info**: Shows local day count and year for real worlds
- **Fictional Systems**: Creative but consistent time systems for fictional worlds

## Default Locations

When users first access the Galactic page, these 4 default locations are shown:
1. **Mars** - The Red Planet (most familiar)
2. **Luna** - Earth's Moon (closest celestial body)
3. **Europa** - Jupiter's ice moon (potential for life)
4. **Proxima Centauri b** - Nearest potentially habitable exoplanet

## User Features

- **Save Locations**: Users can save their favorite galactic locations
- **Custom Selection**: Choose from 22+ real and fictional worlds
- **Categorized Browser**: Organized by Planets, Moons, Exoplanets, and Fictional Worlds
- **Real-time Updates**: All times update every second based on current Earth time
- **Astronomical Accuracy**: Real locations use actual astronomical data from NASA/ESA sources
